# Jogo da Velha da FADAP

Um jogo da velha moderno e elegante, desenvolvido com JavaScript modular, HTML5 e CSS3.

O projeto conta com:
Tabuleiro 3x3 interativo
Alternancia automatica entre jogadores
Verificacao de vitoria e empate
Destaque visual da linha vencedora
Efeito de confete na vitoria
Placar persistente entre rodadas
Modo contra IA inteligente
Design premium com tema escuro

A arquitetura segue principios SOLID para garantir modularidade e facil manutencao.

Como Executar:
1. Clone o repositorio
2. 2. Abra o arquivo index.html no navegador
  
   3. Tecnologias: HTML5, CSS3, JavaScript ES6+
